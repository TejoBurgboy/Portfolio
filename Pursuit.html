Stage Pursuit

In het najaar van 2024 en het begin van 2025 heb ik mijn stage gedaan bij het bedrijf Pursuit. Tijdens deze stage heb ik veel gewerkt met WordPress-websites, waarbij ik gebruik heb gemaakt van WordPress-editors zoals Salient en Elementor.

Salient

Een van de WordPress editors die ik gebruikte, was Salient. Met Salient heb ik gewerkt aan de websites van Elise Mathilde, Buzzzle en Respect Cleaning. Het ontwikkelen van deze websites gebeurde meestal op basis van een aangeleverd design, of zonder design maar dan onder begeleiding. Op de foto links zie je een screenshot van de Elise Mathilde-website, van een onderdeel dat ik heb gemaakt.

Elementor

De andere WordPress-editor die ik gebruikte, was Elementor. De websites die ik heb gemaakt, waren die van Gastronomie Groenenveld en Foresti. Beide sites heb ik gemaakt op basis van een aangeleverd design. Op de foto links is de website van Gastronomie Groenenveld te zien.

CSS

Tijdens mijn stage heb ik ook mijn CSS skills verbeterd. De meeste CSS die ik heb gebruikt, was bedoeld om de responsiviteit te verbeteren en om elementen beter te stylen. Ik heb veel gewerkt met pseudo-elementen en pseudo-classes, zoals ::after en :hover. Een voorbeeld van waar ik met CSS heb gewerkt, is de header van Respect Cleaning, zoals te zien is op de foto links. De positie van de items in de header en de hover effect heb ik met CSS gerealiseerd.

Reflectie

Het grootste doel van deze stage was om mijn CSS skills te verbeteren. Dat is goed gelukt ook heb ik ervaring op gedaan in het werken in een bedrijf en voor een klant. Dit is ook de eerste keer dat ik in sprints heb gewerkt en bij tech meetings ben geweest. Gebruikte skills: Wordpress, Responsieve, Plugins, Design en werken vanuit een design Gebruikte tools: Salient, CSS en Elementor
